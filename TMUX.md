# TMUX Terminal Commands

* Open Tmux session :
 ```
tmux
```

* Split terminal :
```
shift + ctl +b  + % (vertical)
shift +ctl +b + " (horizontal)
  
```

* Navigate in pans
   ```
  ctl + b + o (next)
  ctl + b + l (last)
  ctl + b + ->
  ctl +b + <-

    ```
* Close pan
  ```
  ctl + b + x
  ```
* Close tmux session

```
ctl + b + d
```
